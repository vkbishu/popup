# Popup UI

## Project setup

### Clone and install depencencies
```
git clone https://github.com/vkbishu/popup.git
cd popup
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
